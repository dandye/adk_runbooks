# Tool: get_entities_related_to_a_hunting_ruleset

**MCP Server:** gti-mcp

**Description:** Retrieve entities related to the the given Hunting Ruleset.

**Arguments:**

*   `ruleset_id` (str): Hunting ruleset identifier.
*   `relationship_name` (str): Relationship name.

**Returns:**
